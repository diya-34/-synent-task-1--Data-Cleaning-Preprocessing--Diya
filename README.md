# 🎬 Netflix Exploratory Data Analysis (EDA)

## 📌 Project Overview

This project performs Exploratory Data Analysis (EDA) on the Netflix Titles Dataset to identify trends, patterns, and insights related to Netflix movies and TV shows.

The analysis includes data cleaning, visualization, and trend identification using Python libraries.

## 🎯 Objective

* Analyze Netflix content data
* Identify trends in Movies and TV Shows
* Explore content distribution by country, genre, and release year
* Generate insights through visualizations

## 📂 Dataset

Netflix Titles Dataset (Kaggle)

Dataset Features:

* Show ID
* Type
* Title
* Director
* Cast
* Country
* Date Added
* Release Year
* Rating
* Duration
* Listed In (Genre)
* Description

## 🛠 Tools & Technologies

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Jupyter Notebook

## 📊 Analysis Performed

1. Data Cleaning
2. Missing Value Handling
3. Movies vs TV Shows Analysis
4. Content Added Over Time
5. Top Countries Producing Content
6. Rating Distribution Analysis
7. Genre Analysis
8. Release Year Trends

## 📈 Key Insights

* Movies dominate Netflix content compared to TV Shows.
* The United States contributes the highest number of titles.
* Content production increased significantly after 2015.
* Drama and International Movies are among the most common genres.
